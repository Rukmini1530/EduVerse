# 🐶 EduVerse

EduVerse is an AI-inspired gamified learning platform built using Streamlit.

## Features
- Upload PDF notes
- Automatic quiz generation
- XP and Level system
- Hearts (lives) system
- Daily streak tracking
- Duolingo-style UI

## Tech Stack
- Python
- Streamlit
- PyPDF2

## How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
